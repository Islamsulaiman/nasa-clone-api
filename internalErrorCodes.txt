This file will contain internal error codes in the API to pass errors from the API to the main error function

1. users:

CODE : EXPLAINATION
1 : user not created due to network issue
2 : Email or password dosnt match!, try again
3 : No user values provided for update
4 : User didnt update due to network issues
5 : provided token user not found 


2. search

CODE : EXPLAINATION
11 : the search didnt return data


3. favorites

CODE : EXPLAINATION
21 : if we added the favorite to the same user twice
22 : remove favorite not there in the user
